# 第 7 回 Python 基礎問題

**学習ポイント**：関数

---

## **問題 25: 関数の定義と呼び出し**

**課題:**

引数を取らずに「Hello, Python!」と表示する関数 greet() を定義し、呼び出してください。

**出力例:**

```
Hello, Python!
```

---

## **問題 26: 引数を使う関数**

**課題:**

名前を引数として受け取り、次の形式で挨拶を表示する関数を作成してください。

**出力例:**

```
hello("太郎")
```

出力:

```
こんにちは、太郎さん！
```

---

## **問題 27: 戻り値を持つ関数**

**課題:**

2 つの数を引数として受け取り、その和を返す関数を作成してください。

戻り値を変数に代入し、print で表示してください。

**出力例:**

```
合計は 30 です
```

---

## **問題 28: 関数と計算処理**

**課題:**

商品の価格と個数を引数として受け取り、合計金額を計算して返す関数を作成してください。

**出力例:**

```
360
```
