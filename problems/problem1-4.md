## **問題 3: 計算処理（浮動小数点）**

## **問題 4: レシート風の出力**

**課題:**

次の変数をあらかじめ用意してください。

```
item1 = "りんご"
price1 = 120
qty1 = 2

item2 = "みかん"
price2 = 80
qty2 = 5

tax_rate = 0.1  # 消費税率10%
```

このとき、次の形式で「レシート」を表示してください。

```
---- レシート ----
りんご 2個 x 120円 = 240円
みかん 5個 x 80円 = 400円
小計: 640円
消費税(10%): 64円
合計: 704円
------------------
```
