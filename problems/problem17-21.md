# 第 5 回 Python 基礎問題

**学習ポイント**：リストの基本

---

## **問題 17: リストの作成と表示**

**課題:**

次のリストを作成し、print で表示してください。

- りんご
- みかん
- ぶどう

**出力例:**

```
['りんご', 'みかん', 'ぶどう']
```

---

## **問題 18: 要素へのアクセス**

**課題:**

リスト fruits = ["りんご", "みかん", "ぶどう"] の

1 番目と 3 番目の要素を表示してください。

**出力例:**

```
りんご
ぶどう
```

---

## **問題 19: 要素の追加と削除**

**課題:**

リストを次のように操作してください。

```
fruits = ["りんご", "みかん"]
```

1. "ぶどう" を追加
2. "みかん" を削除
3. 最後にリストを表示

**出力例:**

```
['りんご', 'ぶどう']
```

---

## **問題 20: 合計と平均**

**課題:**

次のリストの合計と平均を計算してください。

```
numbers = [10, 20, 30, 40, 50]
```

**出力例:**

```
合計: 150
平均: 30.0
```
